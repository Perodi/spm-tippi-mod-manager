# Tippi Mod Manager
Currently very early in development. Everything is TBD. Including this README.

## Installation
To use the current development prerelease build. Copy the contents of ```release.zip``` into the riivolution folder in the root of the SD card. On Dolphin locate the it's "Load" directory and extract the contents of the zip file directly in the "Riivolution" directory. Then run and use with the Riivolution patch enabled.

Rename the REL mod you want to use to "mod.rel" and place it in the "spm" directory.

## Build
Requires devkitPPC (see their [getting started](https://devkitpro.org/wiki/Getting_Started) page), and the appropriate build tools (TBD).

- Run ```make preloader```
